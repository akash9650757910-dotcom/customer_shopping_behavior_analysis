# Customer_Shopping_Behavior_Analysis
Data analytics project showcasing customer shopping behavior analysis using python sql and power Bi

## Overview

This project focuses on **Customer Shopping Behavior Analysis**, demonstrating an end-to-end data analytics workflow using real transactional data. The goal is to extract actionable business insights related to **customer spending, subscriptions, discounts, products, and purchasing patterns**.

The project showcases hands-on experience with:

* Data analysis and cleaning using Python
* Writing analytical SQL queries on relational databases
* Building an interactive Power BI dashboard
* Communicating insights through reports and presentations



## Dataset

* **Dataset Name:** Customer Shopping Behavior Dataset
* **Format:** CSV
* **Description:** Contains customer demographics, purchase details, product categories, discounts, subscriptions, shipping types, and review ratings
* **Key Columns:**

  * Customer ID, Gender, Age Group
  * Item Purchased, Category
  * Purchase Amount, Discount Applied
  * Subscription Status, Shipping Type
  * Previous Purchases, Review Rating

The dataset is structured to support **EDA, SQL analysis, and dashboard reporting**.



## Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib/Seaborn)
* **SQL** (PostgreSQL / MySQL / SQL Server)
* **Power BI** (Data modeling, DAX, visualizations)
* **Gamma** (Presentation creation)
* **Jupyter Notebook**


## Project Steps

1. **Data Loading (Python)**

   * Loaded the customer shopping dataset into Python using Pandas

2. **Exploratory Data Analysis (EDA)**

   * Analyzed customer demographics and purchase behavior
   * Studied spending patterns, discounts, and subscriptions
   * Identified missing values and inconsistencies

3. **Data Cleaning & Preparation**

   * Handled missing and invalid values
   * Converted data types for analysis and SQL compatibility
   * Standardized column names

4. **SQL Analysis**

   * Loaded cleaned data into SQL databases
   * Wrote queries to analyze:

     * Revenue by gender and age group
     * Subscriber vs non-subscriber spending
     * Discount usage patterns
     * Top products and categories
     * Customer segmentation (New, Returning, Loyal)

5. **Power BI Dashboard**

   * Built a data model and measures
   * Created interactive visuals and KPIs for business users

6. **Reporting & Presentation**

   * Summarized insights in a structured report
   * Created a professional presentation using Gamma


## Dashboard

The Power BI dashboard (**Customer Behavior Dashboard**) provides:

* Total revenue and average spend metrics
* Revenue breakdown by gender and age group
* Subscriber vs non-subscriber comparison
* Top-performing products and categories
* Discount impact on customer spending
* Interactive slicers for deeper analysis

The dashboard is designed to support **data-driven business decisions**.


## Results & Insights

* Identified key revenue-generating customer segments
* Observed higher average spend among subscribed customers
* Highlighted products frequently purchased with discounts
* Segmented customers into New, Returning, and Loyal groups
* Provided clear insights into customer purchasing behavior
